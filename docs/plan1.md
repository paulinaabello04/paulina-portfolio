# Plan 1 — Portfolio Web (Web Foundations)

## Feature / Tarea
Estructura base del portfolio web y navegación principal.

## Fase 1 — Prompt a la IA
Necesito un plan para crear un portfolio web profesional en HTML, CSS y JavaScript vanilla.
Debe incluir navegación responsive, secciones claras (hero, proyectos, skills, contacto),
uso de CSS custom properties y manipulación básica del DOM sin frameworks.

## Fase 1 — Respuesta de la IA (resumen)
La IA propuso:
- Definir una estructura semántica en HTML
- Crear un sistema de variables CSS para colores, tipografía y espaciados
- Implementar una navegación responsive con menú mobile
- Usar JavaScript para interacciones simples (toggle menú, modal de proyectos)
- Mantener el código modular y legible

## Fase 2 — Implementación
A partir del plan:
- Implementé la estructura HTML manualmente
- Diseñé el sistema visual con CSS custom properties
- Desarrollé la navegación responsive y los modales de proyectos en JavaScript vanilla
- Ajusté estilos, layout y comportamiento según los requisitos del curso

## Notas personales
La IA se utilizó como apoyo conceptual y de planificación.
Todo el código fue entendido, adaptado y escrito conscientemente durante el desarrollo.
